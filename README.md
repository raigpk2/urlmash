# Professional Landing Page Project

This is a production-ready, modularized version of the original single-file HTML landing page. All functionality, design, UI, logic, behavior, and output have been preserved 100%.

## Folder Structure

- `index.html`: Clean HTML structure with references to external assets.
- `css/`: Contains all stylesheets (`style.css`).
- `js/`: Contains modularized JavaScript logic (`main.js`, `player.js`, `ads.js`, `counter.js`, `browser.js`).
- `libs/`: Contains third-party library wrappers (e.g., `t.js` for the analytics widget).
- `assets/`: Reserved for static assets (images, videos, icons).

## Features Preserved

- Video Player with custom play button
- Multiple Video Support & Random Video Selection
- Real-time Visitor Counter (Counter API)
- Ad Rotator with LocalStorage Cooldown Timer
- Back Button Redirect Logic
- Browser & OS Detection (adds classes to `<html>`)
- Whos.amung.us Analytics Widget

## Usage

Simply open `index.html` in a web browser or serve it via a local development server. No build step is required.
